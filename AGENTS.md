# Agents.md Template for Research Projects


## Communication Guidelines

- Refer to the user as **Elia**
- Be concise and implementation-first.
- If a request conflicts with these rules, follow the user and document the exception in `agent_log/INTERACTION_LOG.md`.
- We're economists, not a computer scientists or a programmers. Use precise language and avoid programming jargon. 

---

## Network Access
  - Never upload files online except in the project folder

## Logging (Mandatory)

- Always log all interactions in INTERACTION_LOG.md and index in INDEX.md
- Record a daily summary of activities in human-friendly way in DAILY_LOG.md
